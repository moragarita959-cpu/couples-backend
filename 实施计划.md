# 情侣APP实施计划

## 1. 配置腾讯云开发环境

### 步骤1：注册腾讯云账号
- 访问 [腾讯云官网](https://cloud.tencent.com/)
- 点击「注册」按钮，按照提示完成账号注册
- 完成实名认证

### 步骤2：创建云开发环境
- 登录腾讯云控制台
- 搜索并进入「云开发」服务
- 点击「新建环境」按钮
- 填写环境名称（例如：hao-yi）
- 选择「按量计费」模式
- 点击「立即开通」
- 等待环境创建完成（约1-2分钟）

### 步骤3：获取环境ID
- 在云开发控制台中，找到刚创建的环境
- 复制环境ID（格式类似：hao-yi-2g77ffe4ffd5f3b7）

## 2. 更新云开发环境ID

### 步骤1：修改cloud.js文件
- 打开 `couples-app/utils/cloud.js` 文件
- 将第9行和第16行的 `env` 参数值替换为你的环境ID

## 3. 创建数据库集合

### 步骤1：进入数据库管理
- 在云开发控制台中，点击左侧菜单的「数据库」
- 点击「创建集合」按钮

### 步骤2：创建必要的集合
按照以下列表创建数据库集合：

1. **users** - 存储用户信息
2. **couples** - 存储情侣关系
3. **bind_requests** - 存储绑定请求
4. **messages** - 存储聊天消息
5. **bills** - 存储账单信息
6. **albums** - 存储相册信息
7. **diaries** - 存储日记信息

### 步骤3：配置集合权限
- 为每个集合设置适当的权限
- 建议：
  - users：仅创建者可读写
  - couples：仅创建者和情侣可读写
  - bind_requests：仅创建者可读写
  - messages：仅情侣双方可读写
  - bills：仅情侣双方可读写
  - albums：仅情侣双方可读写
  - diaries：仅情侣双方可读写

## 4. 安装项目依赖

### 步骤1：打开命令行
- 在 HBuilderX 中，点击「终端」→「新建终端」
- 或者在系统中打开命令行工具，导航到项目根目录

### 步骤2：安装依赖
执行以下命令：
```bash
npm install
```

## 5. 运行开发服务器

### 步骤1：在HBuilderX中运行
- 在HBuilderX中打开项目
- 点击顶部菜单的「运行」→「运行到浏览器」→「Chrome」
- 等待开发服务器启动

### 步骤2：查看运行效果
- 浏览器会自动打开应用页面
- 查看是否能正常加载

## 6. 测试应用功能

### 测试登录功能
- 输入手机号和验证码
- 勾选用户协议和隐私政策
- 点击「登录/注册」按钮
- 验证是否能成功登录并跳转到首页

### 测试情侣绑定功能
- 点击首页的「情侣绑定」入口
- 输入对方手机号
- 点击「发送绑定请求」按钮
- 验证是否提示绑定请求已发送

### 测试聊天功能
- 进入聊天页面
- 输入消息并发送
- 验证是否能看到自己发送的消息和模拟的回复
- 测试表情发送功能

### 测试记账功能
- 进入记账页面
- 输入金额、选择类型、分类、日期和描述
- 点击「保存」按钮
- 验证账单是否显示在列表中

### 测试相册功能
- 进入相册页面
- 点击「+ 上传照片」按钮
- 选择一张照片上传
- 验证照片是否显示在相册中
- 测试点赞和评论功能

### 测试日记功能
- 进入日记页面
- 点击「+ 写日记」按钮
- 填写标题、内容、天气和地点
- 点击「保存」按钮
- 验证日记是否显示在列表中
- 测试点赞和评论功能

### 测试设置功能
- 进入设置页面
- 测试通知设置的开关
- 测试编辑纪念日和见面日期
- 测试退出登录功能

## 7. 构建和部署

### 步骤1：构建H5版本
- 在HBuilderX中，点击顶部菜单的「发行」→「网站-H5手机版」
- 填写网站标题为「好意」
- 点击「发行」按钮
- 等待构建完成

### 步骤2：部署H5版本
- 构建完成后，会生成一个 `unpackage/dist/build/h5` 目录
- 将该目录下的所有文件上传到你的服务器或托管平台（如GitHub Pages、Vercel等）

### 步骤3：构建小程序版本
- 在HBuilderX中，点击顶部菜单的「发行」→「小程序-微信」
- 填写小程序AppID（需要在微信公众平台注册）
- 点击「发行」按钮
- 等待构建完成

### 步骤4：部署小程序版本
- 构建完成后，会生成一个 `unpackage/dist/build/mp-weixin` 目录
- 打开微信开发者工具，导入该目录
- 上传代码并提交审核
- 审核通过后发布小程序

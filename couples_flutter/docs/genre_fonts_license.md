# Genre Fonts License

## Notes

- All font files are bundled locally under `assets/fonts/genres/`.
- Genre-specific files such as `GenrePopZh.ttf` and `GenrePopEn.ttf` are hardlinked aliases to a smaller set of downloaded source fonts so the app can register one family per genre without relying on system fonts.
- Google Fonts assets below use the SIL Open Font License 1.1.
- LXGW WenKai uses the SIL Open Font License 1.1 from the official GitHub project.
